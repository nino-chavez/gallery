# Image Assets

## Open Graph Image

**File:** `og-image.jpg`
**Required Dimensions:** 1200x630px (Facebook/Twitter recommended size)
**Format:** JPG (optimized) or PNG
**Max Size:** <300KB for fast loading

### Requirements:
